Formulating the uncertainty set#

The first step is to formulate the uncertainty set. There are multiple options, with their associated input parameters. The uncertainty set can be formulated within the uncertain parameter, which takes in the size of the uncertainty and an uncertainty set as parameters.

Mean Robust uncertainty: \(\{ u = (v_1,\dots,v_K) \mid \sum_{k=1} ^K w_k \| A_k(v_k - \bar{d}_k) \|_p^{\text{power}} \le \rho \}\) - \(K\): int * Number of clusters. Default 1. - data : np.array * Data to be clustered. The \(K\) cluster centers are denoted \(\bar{d}_k\) for \(k = 1, \dots,K\). - \(\rho\) : float, optional
* Ellipsoid scaling. Default 1.0. - \(p\) : integer, optional
* Order of the norm. Default 2. - power: integer, optional * Power of the norm. Default 1. - \(A_k\) : np.array, optional * Scaling matrix for u, for each \(k\). Default identity matrix. - Train: boolean, optional * Whether or not \(A_k\) should be trained using given data. Default True.
[2]:

# size of uncertain parameter
m = 5
# Generate data

                                uncertainty_set = lro.MRO(K = 1, rho=2., data = data, train = False))

Box uncertainty: \(\{ u \mid\| Au + b\|_\infty \leq \rho \}\)

  • \(\rho\) : float, optional

    • Box scaling. Default 1.0.

  • \(A\) : np.array, optional

    • Scaling matrix for u. Default identity matrix.

  • \(b\) : np.array, optional

    • Relocation vector for u. Default None.

  • \(\rho_1\) : float, optional

    • Box scaling. Default 1.0.

  • \(\rho_2\) : float, optional

    • 1-norm scaling. Default 1.0.

  • \(A_1\), \(A_2\) : np.array, optional

    • Scaling matrix for u. Default identity matrix.

  • \(b_1\), \(b_2\) : np.array, optional

    • Relocation vector for u. Default None.

Formulating the Robust Problem#

We can now fomulate the Robust Problem, treating the uncertain parameter as a regular parameter.

Example 1: Affine transformed LP, ellipsoidal uncertainty and mean robust uncertainty.#

We solve the problem

\[\begin{split}\begin{array}{ll} \text{minimize} & c^Tx\\ \text{subject to} & (Pu+a)^Tx \leq 10,\\ \end{array}\end{split}\]

where \(c\), \(P\), and \(a\) are constants, and \(u\) is the uncertain parameter from the ellipsoidal set above, where \(b\) is nonzero.

#restate the ellipsoidal set
ellip_u = lro.UncertainParameter(m,
                                uncertainty_set = lro.Ellipsoidal(p = 2,
                                                                  rho=2., b = -np.mean(data, axis = 0)))
n = 4

# formulate cvxpy variable
x_r = cp.Variable(4)

# formulate constraints

Example 2: Multiple uncertain terms, box uncertainty#

We solve the problem

\[\begin{split}\begin{array}{ll} \text{minimize} & c^Tx\\ \text{subject to} & P(Au+ b)^Tx \leq 10 + 2(Au + b)^Tx\\ & x \geq 0,\quad x \leq 1, \end{array}\end{split}\]

where again \(c\), \(P\), and \(a\) are constants, and \(u\) is the uncertain parameter from a box uncertainty set. We note that the uncertain parameter can appear more than once in the uncertain constraint.

Note that using the box uncertainty set is equivalent to using the ellipsoidal uncertainty set with \(p = \infty\)

Example 3: Multiple uncertain constraints, polyhedral uncertainty#

We solve the problem

\[\begin{split}\begin{array}{ll} \text{minimize} & c^Tx\\ \text{subject to} & (P_1u + a)^Tx \leq 10\\ & (P_2u)^Tx \leq 5\\ & x \leq 5, \end{array}\end{split}\]

where \(c\), \(P_1\), \(P_2\), and \(a\) are constants, and \(u\) is the uncertain parameter from a polyhedral uncertainty set

We compare it with its direct reformulation, where each uncertain constraint is treated independently,

\[\begin{split}\begin{array}{ll} \text{minimize} & c^Tx\\ \text{subject to} & a^Tx + w_1^Td \leq 10\\ & w_1^TD = P_1^Tx\\ & w_2^Td \leq 5\\ & w_2^TD = P_2^Tx\\ & w_1, w_2 \geq 0\\ & x \leq 5, \end{array}\end{split}\]

where \(w_1\), \(w_2\) are new variables introduced.

Example 4: Direct comparison constraints, budget uncertainty#

We solve the problem

\[\begin{split}\begin{array}{ll} \text{minimize} & c^Tx\\ \text{subject to} & (Pu + a)^Tx \leq 10\\ & x \geq P_1u, \end{array}\end{split}\]

where \(c\), \(P\),and \(a\) are constants, and \(u\) is the uncertain parameter from a budet uncertainty set. We can include constraints that relates directly the values of \(x\) and \(u\), as in the second constraint.

We compare this with its direct reformulation,

\[\begin{split}\begin{array}{ll} \text{minimize} & c^Tx\\ \text{subject to} & a^Tx + \rho_1\|y\|_1 + \rho_2\|P^Tx - y\|_\infty \leq 10\\ & -x_i + \rho_1\|z_i\|_1 + \rho_2\|(P_1^T)_i - z_i\|_\infty \leq 0, \quad i = 1,\dots,n \end{array}\end{split}\]

where \((P_1^T)_i\) is the \(i\)’th row of \(P_1\).

Example 5: Quadratic uncertainty, ellipsoidal set#

We solve the problem

\[\begin{split}\begin{array}{ll} \text{minimize} & \tau\\ \text{subject to} & -(1/2)\sum_{i=1}^n (u^TP_iu)x_i \leq \tau\\ & \mathbf{1}^Tx = 3\\ & x \geq 0, \quad x \leq 1 \end{array}\end{split}\]

where each \(P_i\in \mathbf{R}^{n\times n}\) is a symmetric positive definite matrix, and \(u\) is the uncertain parameter from an ellipsoidal uncertainty set.

[16]:

# formulate uncertainty set
n = 5
u = lro.UncertainParameter(n,
                        uncertainty_set=lro.Ellipsoidal(p=2,
                                                    rho=0.5))
# formulate cvxpy variables
x_r = cp.Variable(n)
t = cp.Variable()

# formulate problem constants
P = {}
P_inv = {}
for i in range(n):
    P[i] = datasets.make_spd_matrix(n, random_state=i)
    P_inv[i] = sc.linalg.sqrtm(np.linalg.inv(P[i]))

# formulate objective
objective = cp.Minimize(t)

# formulate constraints
constraints = [cp.sum([-0.5*lro.quad_form(u, P[i]*x_r[i]) for i in range(n)]) <= t]
constraints += [cp.sum(x_r)==4]
constraints += [x_r >= 0.6, x_r <= 1]

# formulate Robust Problem
prob_robust = lro.RobustProblem(objective, constraints)

# solve
prob_robust.solve()
print("LRO objective value: ", prob_robust.objective.value, "\nLRO x: ", x_r.value)

LRO objective value:  -2.001521732684513e-10
LRO x:  [0.79877228 0.80135493 0.80078944 0.79878038 0.80030297]
We compare this with the direct reformulation, :nbsphinx-math:`begin{equation}
begin{array}{lll}

&text{minimize} & tau \ &text{subject to} & (1/2)sum_{i=1}^n (z_i^TP_i^{-1}z_i)/x_i + rho|z|_2 le tau\ && sum_{i=1}^n z_i = z,\

&& mathbf{1}^Tx = 3\ && x geq 0, quad x leq 1

end{array}

end{equation}` where we have \(\gamma \in \mathbf{R}^m, z \in \mathbf{R}^n, z_i \in \mathbf{R}^n\), \(i = 1,\dots,n\), as auxiliary variables.

Example 6: Max of affine uncertainty, ellipsoidal set, mean robust set#

We solve the problem

\[\begin{split}\begin{array}{ll} \text{minimize} & \tau \\ \text{subject to } & \max\{a^Tx - d^Tx, a^Tx - d^T(3u+3)\} \leq \tau,\\ \end{array}\end{split}\]

where \(a\) and \(d\) are constants, and \(u\) is the uncertain parameter from an ellipsoidal uncertainty set.
